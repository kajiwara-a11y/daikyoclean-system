# ダイキョウクリーン 業務管理システム

株式会社ダイキョウクリーン向け 業務管理システム（プロトタイプ）。
PC向けの管理ダッシュボードと、営業／現場向けのスマホアプリ（操作版）で構成。

## 構成

| ファイル / ディレクトリ | 内容 |
|---|---|
| `index.html` | PC向け 業務管理ダッシュボード（顧客・契約・作業・配車・請求・入金・権限・BI など） |
| `ops.html` | モバイル向け 営業／現場アプリ（操作版） |
| `js/` | 画面・ロジック（バニラ JS）。画面は `scr_xxx()`、遷移は `route(id,tab)` / モバイルは `setState`・`MDATA` |
| `fonts/` | 同梱フォント（Noto Sans JP サブセット） |

## ローカルで確認

```bash
npx http-server . -p 3457 -c-1
# → http://localhost:3457/        （ダッシュボード）
# → http://localhost:3457/ops.html（スマホアプリ）
```

## デプロイ（Vercel / ALION team）

```bash
npx vercel deploy . --prod --scope alion
```

本番URL: https://daikyoclean-system.vercel.app
